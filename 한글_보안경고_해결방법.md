# 한글(호글) 파일 접근 보안 경고 해결 방법

앱에서 품위서·일일활동일지·계획서를 만들 때 한글이 **「위 파일에 접근하려는 시도가 있습니다」** 라는 창을 띄우는 경우가 있습니다.

---

## 1. 지금 당장 쓰려면 (가장 간단)

경고창이 뜨면:

- **「모두 허용(N)」** 을 누르세요.  
  → 앞으로 같은 PC에서 이 앱이 HWP 파일을 열거나 저장할 때 같은 경고가 뜨지 않습니다.
- 또는 **「접근 허용(Y)」** 만 눌러도 이번 한 번은 진행됩니다. (다음에 또 창이 뜰 수 있음)

---

## 2. 한글에서 “신뢰할 수 있는 경로”로 추가 (권장)

한글을 **먼저 실행**한 뒤:

1. **파일** → **환경 설정** (또는 **도구** → **환경 설정**)
2. **보안** 또는 **신뢰할 수 있는 경로** 메뉴로 이동
3. 다음 폴더를 **추가**:
   ```
   C:\Users\user\Desktop\방과후센터앱\app\outputs
   ```
4. 저장 후 한글을 다시 실행해 보세요.

이렇게 하면 이 폴더에 있는 HWP 파일에 대한 접근 경고가 줄어들 수 있습니다.

---

## 3. 보안 모듈 등록 (고급)

hwp-mcp의 보안 DLL을 사용 중이라면, 한글이 해당 모듈을 쓰도록 **레지스트리**에 등록할 수 있습니다.

1. **Win + R** → `regedit` 입력 후 실행
2. 다음 경로로 이동:
   ```
   HKEY_CURRENT_USER\Software\HNC\HwpAutomation\Modules
   ```
3. 오른쪽 창에서 **새로 만들기** → **문자열 값**
4. 이름: `SecurityModule`
5. 값: 보안 모듈 DLL 전체 경로  
   (예: `C:\Users\user\AppData\Roaming\Cursor\mcp-servers\hwp-mcp\security_module\FilePathCheckerModuleExample.dll`)  
   **경로에 따옴표 없이** 입력
6. 레지스트리 편집기 종료 후 **한글을 완전히 종료했다가 다시 실행**

---

정리하면, **일단은 경고창이 뜰 때마다 「모두 허용(N)」만 눌러 주시면** 앱을 계속 사용하실 수 있습니다.
