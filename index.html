올리브청소년방과후 센터 활동앱

GitHub Pages 한계: 이 주소는 정적 웹페이지만 보여 줄 수 있습니다. 본 앱은 Flask(Python) 서버가 필요하므로, 여기서는 실제 활동앱 화면이 열리지 않습니다.

소스 코드와 설치 방법은 GitHub 저장소를 참고하세요.

PC에서 실행 (요약)

git clone https://github.com/slitlamp2/olive-after-school-app.git
cd olive-after-school-app
python -m venv .venv
.venv\Scripts\activate
pip install -r app/requirements.txt
# app/.env 에 OPENAI_API_KEY 등 설정
cd app
python app.py

브라우저에서 http://localhost:5006 로 접속합니다.
