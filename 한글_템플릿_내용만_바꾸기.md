# 예시 한글 파일로 표·형식 유지하고 내용만 바꾸는 방법

예시로 만든 한글 파일(템플릿)을 열어서 **표와 서식은 그대로 두고, 글자만 바꾸는** 방법입니다.  
Cursor 채팅에서 HWP MCP를 쓰면 AI가 아래 도구들을 조합해서 처리해 줍니다.

---

## 1. 전체 흐름 요약

1. **템플릿 한글 파일**을 정해진 폴더에 둡니다. (예: `C:\Users\윤미란\Desktop\방과후센터\템플릿\일일활동일지.hwp`)
2. **hwp_open(경로)** 로 그 파일을 엽니다.
3. **본문(표 밖) 글만 바꿀 때** → **hwp_replace_text**(찾을 글, 바꿀 글)
4. **표 안 셀 내용만 바꿀 때** → **hwp_find_and_show_cell** + **hwp_table_view** + **hwp_fill_cells**
5. **hwp_save(새 경로)** 로 “다른 이름으로 저장”해서 새 파일로 남깁니다.

---

## 2. 본문 텍스트만 바꾸기 (표 밖 문단/문장)

**도구**: `hwp_replace_text`

- **find**: 바꿀 **원본 문구** (템플릿에 들어 있는 그대로 적기)
- **replace**: 넣을 **새 문구**
- **replace_all**: `true`면 문서 전체에서 같은 문구를 한 번에 모두 바꿈 (보통 true 사용)

같은 문구가 여러 번 나오면 전부 바뀌므로, **고유한 문구**로 찾는 것이 안전합니다.  
(예: `"[활동내용]"` → `"미술 활동 진행"`)

**채팅 예시**  
*「한글 파일 C:\...\일지_템플릿.hwp 열고, "[날짜]" 를 "2025-03-02" 로, "[활동내용]" 을 "미술 활동" 으로 바꾼 다음 바탕화면에 일지_0302.hwp 로 저장해 줘」*

---

## 3. 표 안 셀 내용만 바꾸기 (표·형식 유지)

표는 **그대로 두고** 특정 셀의 **글만** 바꿀 때 쓰는 순서입니다.

### 3-1. 표 구조 파악하기

1. **hwp_find_and_show_cell("레이블")**  
   표 안에 있는 **고유한 글자**(예: "이름", "날짜", "품목")를 넣으면, 그 텍스트가 있는 셀로 커서가 이동하고 그 셀 내용을 알려 줍니다.  
   → “이 셀을 기준으로 오른쪽/아래 셀에 값을 채우자” 할 때의 **시작점**입니다.

2. **hwp_table_view(depth=2)**  
   현재 커서가 있는 셀을 중심으로 **위/아래/좌/우** 셀 내용을 보여 줍니다.  
   → 표가 몇 행 몇 열인지, 헤더가 어디인지 파악할 때 사용합니다.

3. (선택) **hwp_navigate("left" | "right" | "up" | "down")**  
   표 안에서 한 칸씩 이동하면서 셀 내용을 확인할 때 씁니다.

### 3-2. 표 셀에 값 넣기: hwp_fill_cells

**도구**: `hwp_fill_cells`

- **path_value_map**: “어느 셀에 무엇을 쓸지”를 **경로 → 값** 형태로 여러 개 넣습니다.
  - **경로 형식**: `"찾을텍스트 > <right> > <down>"` 처럼, **먼저 찾을 텍스트**를 쓰고, 그 다음 **이동 방향**을 나열합니다.
  - 방향: `<left>`, `<right>`, `<up>`, `<down>`
- **mode**: `"replace"`(기본, 기존 내용 지우고 입력), `"prepend"`(앞에 추가), `"append"`(뒤에 추가)

**예시**

- "이름" 이 적힌 셀 **오른쪽** 칸에 "홍길동" 넣기:  
  `{"이름 > <right>": "홍길동"}`
- "날짜" 오른쪽에 "2025-03-02" 넣기:  
  `{"날짜 > <right>": "2025-03-02"}`
- 여러 셀 한 번에:  
  `{"이름 > <right>": "홍길동", "날짜 > <right>": "2025-03-02", "활동내용 > <right>": "미술 활동"}`

**채팅 예시**  
*「지금 연 한글 문서 표에서 "이름" 오른쪽 셀에 "김철수", "날짜" 오른쪽 셀에 "2025-03-02" 넣어 줘」*  
→ AI가 `hwp_fill_cells`에 위처럼 path_value_map을 넣어 호출하면 됩니다.

---

## 4. 한 번에 하기: 템플릿 열기 → 본문·표 내용만 수정 → 저장

**권장 순서**

1. **hwp_open**(템플릿 파일 전체 경로)
2. **hwp_replace_text**로 본문에 있는 플레이스홀더/예시 문구만 치환 (필요한 만큼 반복)
3. **hwp_fill_cells**로 표 셀만 원하는 값으로 채우기 (표 구조는 건드리지 않음)
4. **hwp_save**(새 파일 경로) → 형식·표 그대로, 내용만 바뀐 새 한글 파일 생성

**채팅 예시 (한 문장으로)**  
*「C:\Users\윤미란\Desktop\방과후센터\템플릿\일일활동일지.hwp 열고, [날짜]를 2025-03-02로, [학생이름]을 김철수로, 표에서 "활동내용" 오른쪽 셀에 "미술 활동" 넣은 다음, 바탕화면에 일지_0302_김철수.hwp 로 저장해 줘」*

---

## 5. 실무 팁

- **템플릿에 플레이스홀더 넣어 두기**  
  나중에 바꿀 자리는 `[날짜]`, `[학생이름]`, `[활동내용]` 처럼 **고유한 표시**를 해 두면, `hwp_replace_text`로 정확히 그 부분만 바꾸기 쉽습니다.

- **표는 “레이블”을 기준으로**  
  표 첫 행이나 첫 열에 "이름", "날짜", "품목" 같은 **고정 레이블**을 두면, `hwp_find_and_show_cell` + `hwp_fill_cells`로 “레이블 > <right>” 같은 경로만 지정해도 셀을 안전하게 채울 수 있습니다.

- **저장은 항상 새 경로로**  
  원본 템플릿은 덮어쓰지 말고, `hwp_save(새 경로)` 로 “다른 이름으로 저장”하는 습관을 두면, 같은 템플릿을 계속 재사용할 수 있습니다.

---

## 6. 사용하는 HWP MCP 도구 정리

| 목적 | 도구 |
|------|------|
| 템플릿 열기 | **hwp_open**(path) |
| 본문에서 글만 바꾸기 | **hwp_replace_text**(find, replace, replace_all) |
| 표에서 “이 셀” 찾기 | **hwp_find_and_show_cell**(text) |
| 표 구조 보기 | **hwp_table_view**(depth) |
| 표 셀에 값 넣기 (형식 유지) | **hwp_fill_cells**(path_value_map, mode) |
| 표 안에서 한 칸 이동 | **hwp_navigate**(direction) |
| 내용만 바꾼 새 파일로 저장 | **hwp_save**(path) |

---

정리하면, **예시 한글 파일을 올려 두고** 그 경로를 알려 주면, **표와 형식은 그대로 두고 내용만 바꾸는** 작업은 위 순서대로 `hwp_open` → `hwp_replace_text` / `hwp_fill_cells` → `hwp_save` 로 처리할 수 있습니다.
